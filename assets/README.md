# Assets Folder

This folder contains branding assets for the Streamlit UI.

## Logo

- **Current**: `rsm_logo.svg` - SVG version of RSM logo
- **To Replace**: If you have the actual RSM logo file (PNG, SVG, or JPG), replace `rsm_logo.svg` with your logo file
- **Supported Formats**: SVG (recommended), PNG, JPG
- **Recommended Size**: 120x40 pixels or similar aspect ratio

## How to Replace Logo

1. Place your logo file in this `assets/` folder
2. Name it `rsm_logo.svg` (or update the path in `app.py` if using a different name)
3. The app will automatically load and display your logo

## Notes

- SVG format is recommended for crisp display at any size
- If using PNG/JPG, ensure it has a transparent background for best results
- The logo is displayed in the header of the application
